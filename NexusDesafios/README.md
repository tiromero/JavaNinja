# NexusDesafios

Espaco separado para desafios e exercicios extras sugeridos pelo assistente.

Esta pasta nao faz parte da progressao oficial do curso JavaNinja e esta ignorada pelo Git.

## Organizacao

- `desafios/`: desafios praticos maiores, geralmente com um objetivo especifico.
- `exercicios/`: exercicios curtos para reforcar conceitos.
- `revisoes/`: anotacoes de revisao, correcoes e aprendizados.

## Regra de uso

Use esta pasta para testar ideias, errar, refazer e registrar aprendizados sem misturar com os exercicios do curso.
